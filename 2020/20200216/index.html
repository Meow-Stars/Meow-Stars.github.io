【iOS】字典中使用自定义结构体（可哈希）

Written by 云深隐星澜 with ♥ on February 16, 2020 in iOS•Swift

自定义了一个结构体Dimension

struct Dimension {
    var dimName: String?
    var dimWeight: Int?         
}

用作字典中的Key

var score: [Dimension?: Int?]

好了，报错

Dictionary' requires that 'Dimension' conform to 'Hashable

Type 'Dimension' does not conform to protocol 'Hashable'

提取关键字“Hashable”，使用自定义类型作为合集的值类型或者字典的键类型，需要遵循Hashable协议。

Hashable协议遵循Equatable，需要实现 ==，即等价关系。等价关系必须满足以下三个条件，其中 a, b, c是任意值：

  • 自反性：a == a
  • 对称性：a == b 等价于 b == a
  • 传递性：a == b && b == c 等价于 a == c

解决

extension Dimension: Hashable{          // 使该类型可哈希
    static func == (dim1: Dimension, dim2: Dimension) -> Bool{
        return dim1.dimName == dim2.dimName && dim1.dimWeight == dim2.dimWeight
    }
    func hash(into hasher: inout Hasher){
        hasher.combine(dimName)
        hasher.combine(dimWeight)
    }
}
