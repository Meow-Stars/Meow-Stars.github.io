【iOS】Swift隐藏键盘

Written by 喵星澜 with ♥ on January 21, 2020 in iOS•Swift

在开发过程中，经常会遇到需要使用文本输入框的情况，一般点击输入框就会弹出键盘，但是键盘的回收就很不友好了，网上找了找还都是Swift3时代的代码，和现在的Swift5根本就是天差地别，简直就是在欺负我们纯纯的小白。

点击空白隐藏键盘

注意把ViewController替换成自己的ViewController名称，不然不会生效哦。

import UIKit
class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        // 点击空白隐藏键盘
        let tap = UITapGestureRecognizer(target: self, action: #selector(ViewController.hideKeyboard(tapG:)))
        tap.cancelsTouchesInView = false
        self.view.addGestureRecognizer(tap)
    }
    // MARK: 隐藏键盘方法实现
    @objc func hideKeyboard(tapG:UITapGestureRecognizer){
        self.view.endEditing(true)
    }
}

备注

  1. “#”后面的内容不是注释哦，只有“//”后面的内容是注释
  2. “// MARK: ”超好用，就像是目录一样，点击顶部文件名后面一个图标，就会出现MARK标注的注释
