Hugo搭建个人博客（三）—— Github绑定顶级域名

Written by 喵星澜 with ♥ on June 27, 2019 in

域名注册

这个就不细讲了，国内阿里云、腾讯云都挺好的，随便挑吧，看哪个优惠力度大。

我选的阿里云，因为.info域名阿里云第一年更便宜。

域名解析

域名注册好后，要将域名解析到博客上才能使用。

  • 进入阿里云域名服务-域名列表。这里会显示在阿里云注册的域名
  • 选择注册好的域名，最后操作一列有解析选项，点击
  • 选择添加记录
  • 在终端输入ping yourName@github.io，可以查看到IP为185.199.110.153。（似乎现在所有Github Page的IP都是这个，不过还是自己ping一遍吧）
  • 添加两个类型为A的记录，一个主机记录为www，另一个为@。记录值都是IP地址

修改Github仓库的CNAME文件

  • 进入yourName.github.io仓库，找到文件CNAME
  • 如果没有，就自己创建一个。选择右上角Create new file
  • 文件名为CNAME，没有文件类型后缀
  • 输入内容为注册好的域名
  • 提交保存后就大功告成！等待几分钟使域名生效，就可以通过顶级域名访问博客了

域名备案

服务器在国内的顶级域名都需要备案，我没用服务器，就先挖个坑，等我哪天开始折腾云服务器了就来填坑。
