Sketch安装插件

Written by 云深隐星澜 with ♥ on October 2, 2019 in

插件安装

Sketch开放了插件系统，有很多优秀插件可以丰富Sketch的功能。

那么问题来了，总要先会安装插件才能享受插件带来的丰富扩展吧︿(￣︶￣)︿

一般Sketch安装插件有两种方法：

  1. 通过第三方插件管理工具安装
  2. Github下载后手动安装

Sketch插件管理工具

  1. Sketch Toolbox
  2. Sketch Plugin Manager
  3. Sketchpacks
  4. Sketch Runner

Github下载后手动安装

  • 从Github下载插件到本地
  • 解压缩，最后的后缀名应该是.sketchplugin
  • 打开Sketch
  • 在Finder中双击.sketchplugin文件
  • Sketch提示成功安装插件
  • 在插件→管理插件列表中可以进行插件管理
