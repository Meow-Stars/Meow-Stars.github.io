Hugo搭建个人博客（一）—— 本地运行

Written by 云深隐星澜 with ♥ on August 25, 2019 in

前言

搭建个人博客这个想法，最先起源于Ulysess更新，它告诉我，可以直接在Ulysess中发布Ghost博客，我想既然订阅了Ulysess，总要让它物尽其用，然后就兴致冲冲地开始查找资料。 然后发现，Ghost的数据库要求是MySQL5，然而几个月前我做数据库作业时，毫不犹豫安装了MySQL8.0.15，想想算了吧，于是又把目光瞄向了WordPress。 在挑选WordPress主题时，这款Hugo的主题LeaveIt惊艳了我，于是使用Hugo搭建博客这件事，就这么愉快地决定了。

开始

安装

我用的是MAC OS系统，以前安装过homebrew，所以安装Hugo非常简单，直接打开终端，输入：

brew install hugo

可以通过hugo version语句查看Hugo版本。

建立新站点

  • 这会在/Users/apple创建一个名为MySite的文件夹，这就是新的Hugo站点
hugo new site MySite
  • 进入站点根目录
cd MySite
  • 创建新文章about.md
hugo new about.md

about.md会出现在MySite/content目录下，打开文件，在里面随便写点内容后保存。安装好主题后，这就是About模块的内容

  • 创建另一篇文章first.md
hugo new posts/first.md

这会在MySite/content目录下创建一个新的文件夹posts，以后的博文都存储在这个文件夹下。 同样打开first.md，在其中添加一些内容。

  • 可以发现文章顶部有一些meta信息，其中draft: true表示该篇文章为草稿，在发布时要把true改为false，否则只能在本地运行时查看，不会发布到网站上。

安装主题

  • command+N，新建一个终端界面，进入themes目录
cd themes
  • 然后输入下方代码下载主题LeaveIt
git clone https://github.com/liuzc/LeaveIt.git

  • 下载完成后发现MySite/themes文件夹下多了一个文件夹LeaveIt，点进去，找到exampleSite/config.toml，复制全部内容，然后粘贴到MySite/config.toml中去。这个config.toml是和themes文件夹同级的那个。

  • 在MySite/static下创建新文件夹images

  • 在images下创建新文件夹me

  • 最后找一张你喜欢的图片作为你的大头贴。 最后这张图片的目录是这样子滴：MySite/static/images/me/xxxx.jpeg。这个xxxx.jpeg就是图片名称。

  • 修改config.toml中[params]模块中的avatar信息，将其中最后的图片名称改为自己的。

avatar = "/images/me/xxxx.jpeg"

本地预览

  • 重新打开一个终端界面，进入站点根目录
cd MySite
  • 运行服务器
hugo server -D
  • 打开浏览器，输入http://localhost:1313，就可以看到本地搭建的博客了。

在Hugo服务器运行期间，会实时监测content文件夹中文章内容的改动，网页会自动刷新显示最新修改内容。

在终端界面按下快捷键control+C，可以停止运行服务器。服务器停止运行后，访问localhost:1313会被拒绝。
