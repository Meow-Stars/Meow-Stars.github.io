MAC在Finder中显示隐藏文件

Written by 云深隐星澜 with ♥ on September 1, 2019 in

显示隐藏文件

在终端中输入

defaults write com.apple.Finder AppleShowAllFiles YES
KillAll Finder

效果如图：

不显示隐藏文件

defaults write com.apple.Finder AppleShowAllFiles NO
KillAll Finder
